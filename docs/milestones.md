# Milestones

## Phase 1: MVP Concordance

- Raw PDF/source ingestion.
- Source metadata and citation storage.
- Curated text units.
- Sentence/token splitting.
- Normalized search.
- KWIC frontend.
- Source library and context view.

## Phase 2: Dictionary / Lexicon

- Import legacy lexicon data.
- Expand dictionary entry pages.
- Link corpus tokens to entries/forms through attestations.
- Add source-specific definitions and examples.

## Phase 3: Researcher Tools

- Advanced filters.
- Frequency views.
- CSV/JSON export.
- Correction workflow beyond Django admin.
- Source comparison.

## Phase 4: NLP

- Variant mapping.
- Lemmatization.
- Morphological analysis.
- IPA/pronunciation and audio.
- Confidence/uncertainty workflows for analyses.
